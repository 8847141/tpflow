> ## 开发计划
 1、增加更多的API调用接口（如调用工作流当前状态，审批记录）；

 2、终止工作流，剥离工作流等功能；

 3、减少TP依赖性，为composer做准备；

 4、UI全面优化，组件独立；


---
**TpFlow** 是一款thinkphp5.1/php的工作流插件**

**TpFlow** 可以可视化自定义工作流，支持工作流的会签等

**TpFD**  表单自定义功能

---

### 插件要求

1. PHP版本大于5.6 ，如TP3，需要自行调整部分
2. Tp语法详见thinkphp5
3. 使用本插件需要有一定的Tp基础
